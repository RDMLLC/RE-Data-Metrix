## Fix 1 — Back button not working

The current `window.history.back()` call does nothing when the user navigated directly to the page with no prior history.

Replace the back button logic on all four pages with this pattern:

```tsx
import { useLocation } from 'wouter';

const [, navigate] = useLocation();

const handleBack = () => {
  if (window.history.length > 1) {
    window.history.back();
  } else {
    navigate('/toolbox');
  }
};
```

The back button should call `handleBack()` on click. This way it goes back if there is history, and falls back to `/toolbox` if the user landed directly on the page.
