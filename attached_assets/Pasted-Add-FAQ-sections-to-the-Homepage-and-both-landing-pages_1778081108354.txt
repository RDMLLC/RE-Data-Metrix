Schema markup:
For each page, add a FAQPage JSON-LD script via useEffect (same pattern as BlogPost.tsx Article schema):
tsxuseEffect(() => {
  const schema = {
    "@context": "https://schema.org",
    "@type": "FAQPage",
    "mainEntity": pageFaqs.map(faq => ({
      "@type": "Question",
      "name": faq.question,
      "acceptedAnswer": {
        "@type": "Answer",
        "text": faq.answer
      }
    }))
  };
  const script = document.createElement("script");
  script.type = "application/ld+json";
  script.id = "faq-schema-[pagename]";
  script.textContent = JSON.stringify(schema);
  document.head.appendChild(script);
  return () => {
    const existing = document.getElementById("faq-schema-[pagename]");
    if (existing) existing.remove();
  };
}, []);
Use id="faq-schema-home" for Home.tsx and id="faq-schema-landing" for MetaOffer.tsx.
